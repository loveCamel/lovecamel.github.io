Подключение

Для установки слайдера необходимо подключить js и css файлы

Подклчюение css

Необходимо подключить файл vanilaSlider.min.css внутрь тега <head> </head> на необходимых вам страницах

<link rel="stylesheet" href="vanilaSlider/vanilaSlider.min.css">

Подклчюение js

Необходимо подключить файл vanilaSlider.min.js после тега <body> </body> на необходимых вам страницах

<link rel="stylesheet" href="vanilaSlider/vanilaSlider.min.css">

Параметры

Название Тип Описание
slider Строка CSS селектор слайдера. Используется метод querySelector. Обязательный параметр
slides Строка CSS селектор слайдов. Используется метод querySelectorAll. Обязательный параметр
showCount Число Количество слайдов на экране
btnPrev Строка Селектор кнопки "предыдущий слайд". Если оставить пустым, то появится кнопка по умолчанию
btnNext Строка Селектор кнопки "следующий слайд". Если оставить пустым, то появится кнопка по умолчанию

Методы

Название Описание
nextSlide() Переключает слайд на следующий. Если слайд последний, переключит на первый слайд
prevSlide() Переключает слайд на предыдущий. Если это первый слайд, переключит на последний
autoSlide(time) Включает автоматическое слайдшоу. time - число, время переключения слайда. 1000 - 1 секунда
stopAutoSlide() Останавливает автоматическое слайдшоу
