LightZoom - зум картинок на чистом javaScript

Подключение

Для подключения зума необходимо подключить js и css файлы

Подключение css

Необходимо подключить файл LightZoom.css внутрь тега <head> </head> на необходимых вам страницах

Файл находится по пути: LightZoom\dist\css

Пример подключения:


<link rel="stylesheet" href="LightZoom.css">

Подключение js

Необходимо подключить файл LightZoom.js после тега <body> </body> на необходимых вам страницах

Файл находится по пути: LightZoom\dist\js

Пример подключения:


<script type="text/javascript" src="LightZoom.js"></script>

Опции

Всё опции являются необязательными, т.к имеют значения по умолчанию

Название Тип Описание
selector String Селектор изображений, которые нужно увеличить по клику. Используется метод querySelectorAll.
У изображений должен быть задан атрибут src
По умолчанию: 'img'
isFix Boolean Запрещать ли скролл при открытии модального окна.
По умолчанию: false
isCursorPointer Boolean Показывать ли иконку зума при наведении на картинки.
По умолчанию: false
isOutsideClose Boolean Будет ли модальное окно закрываться при клике вне картинки.
По умолчанию: false
bgColor String Цвет фона(подложки) модального окна. Использовать только полный (6-ти знанчый) формат HEX, примеры:

'#ffffff'
'#eeeeee'
'#24292e'

По умолчанию: '#ffffff'
bgOpacity Number Число от 0 до 100. Прозрачность фона(подложки) модального окна. Где 100, значит полная заливка фона, а 0 - полная прозрачность
По умолчанию: 80

Методы

Название Описание
closeModal() Метод закрывает модальное окно
removeEvents() Метод удаляет все обработчики событий со всех элементов.

Примеры

Минимальные настройки

Вёрстка


<img src='img.jpg' alt='image' class='example__img'>

JavaScript


new LightZoom({
  selector: '.example__img'
});

Запрет на скролл + иконка зума + закрытие при клике мимо картинки

Вёрстка


<img src='img.jpg' alt='image' class='example__img'>

JavaScript


new LightZoom({
  selector: '.example__img',
  isFix: true,
  isCursorPointer: true,
  isOutsideClose: true
});

Непрозрачный синий фон

Вёрстка


<img src='img.jpg' alt='image' class='example__img'>

JavaScript


new LightZoom({
  selector: '.example__img',
  bgColor: '#0067c9',
  bgOpacity: 100
});
